## 项目介绍
基于springboot的学生信息管理系统：前端 html、jquery、layui，后端 maven、springmvc、spring、mybatis；角色分为：管理员、教师、学生；集成学生、班级列表、成绩统计列表等功能于一体的系统。

## 功能介绍

- 基本功能：登录、注册、退出
- 学生列表：学生信息的增删改查，照片上传
- 班级列表：班级信息的增删改查
- 用户列表：用户信息的增删改查
- 成绩统计：我的成绩，班级成绩的增删改查
- 权限管理：所有角色一览，角色权限一览，新增权限角色，授予用户权限
